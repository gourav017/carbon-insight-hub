# Carbon Insight Hub

A comprehensive sustainability platform featuring ESG Assessment, Carbon Emissions Calculator, and Carbon Footprint Analysis tools.

## Features

- **ESG Assessment**: Evaluate Environmental, Social, and Governance performance.
- **Carbon Emissions Calculator**: Calculate Scope 1, 2, and 3 emissions.
- **Carbon Footprint**: Analyze energy, transport, and sector-specific carbon footprints.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (for data persistence)

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/Pranay1301/carbon-insight-hub.git
   ```

2. Navigate to the project directory:
   ```sh
   cd carbon-insight-hub
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

## Deployment

Build the project for production:

```sh
npm run build
```
